## 11.4 框架版本与 API 参考

本附录列出书中涉及的框架版本和 API 信息。

###  模型 API

### OpenAI

| 模型 | 上下文窗口 | 特点 |
|------|------------|------|
| gpt-4-turbo | 128K | 通用能力强 |
| gpt-4o | 128K | 多模态支持 |
| gpt-4o-mini | 128K | 性价比高 |
| o1-preview | 128K | 强推理能力 |

**API 基础用法**：
```python
from openai import OpenAI
client = OpenAI()
response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello"}]
)
```

### Anthropic Claude

| 模型 | 上下文窗口 | 特点 |
|------|------------|------|
| claude-3-5-sonnet | 200K | 编程能力强 |
| claude-3-opus | 200K | 最高智能 |
| claude-3-haiku | 200K | 快速响应 |

**API 基础用法**：
```python
import anthropic
client = anthropic.Anthropic()
message = client.messages.create(
    model="claude-3-5-sonnet-20241022",
    max_tokens=1024,
    messages=[{"role": "user", "content": "Hello"}]
)
```

###  框架版本

- **当前版本**: 0.3.x
- **Python**: 3.9+
- **安装**: `pip install langchain langchain-openai`

### LlamaIndex

- **当前版本**: 0.11.x

- **安装**: `pip install llama-index`

### AutoGen

- **当前版本**: 0.4.x

- **安装**: `pip install autogen-agentchat`

### CrewAI

- **当前版本**: 0.76.x

###  版本兼容性说明

> ⚠️ 本书基于 2025-2026 年的框架版本编写。
> AI 框架更新迅速，建议查阅官方文档获取最新信息。

**常见版本问题解决**：
1. API 变更：查看框架的 CHANGELOG
2. 依赖冲突：使用虚拟环境隔离
3. 功能缺失：确认使用正确的包名（如 langchain-openai vs langchain）
