## 本章小结

本章构建了智能体的记忆系统架构，从认知模型到工程实践。核心路径：短期记忆管理(上下文窗口优化) → 长期记忆存储(向量数据库) → 高级检索技术(RAG) → 结构化记忆(知识图谱) → 上下文工程系统设计 → 实战案例(autoDream)。记忆系统让智能体具备持续学习和个性化能力，成为真正的智能伙伴。

### 关键概念清单

* **认知模型**：短期记忆(工作记忆) vs 长期记忆(情景/语义)
* **向量数据库**：托管型/开源/嵌入式的选择与权衡
* **RAG 进阶**：文档切分、混合检索、重排序的组合优化
* **图记忆**：知识图谱赋予结构化推理能力
* **上下文工程**：滑动窗口、摘要压缩、动态分配的信息最大化
* **autoDream 引擎**：后台记忆整合、多层发现链、向量嵌入语义匹配
