## 框架选型速查表

| 目标场景 | 推荐形态 | 核心优势 |
|:---------|:---------|:---------|
| 快速原型开发 | 角色任务编排 / 托管式接口 | 上手快、集成度高 |
| RAG/知识智能体 | 数据/RAG 驱动 | 索引、检索与后处理能力强 |
| 精细控制流程 | 图编排/状态机 | 分支/循环/检查点可控 |
| 多智能体协作 | 对话编排 + 评审回路 | 分工与验证机制清晰 |
| 企业级集成 | 企业级 SDK/连接器 | 权限、审计与系统集成 |

下一章将探讨如何将智能体系统工程化落地，实现生产级别的稳定运行。
