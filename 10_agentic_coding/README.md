本章将探讨 AI 智能体如何彻底改变软件开发的范式。我们不只是讨论代码生成，而是深入到 **Agentic Coding** 的核心——让 AI 作为一个自主的智能体，参与到理解需求、规划架构、编写代码、调试修复的全生命周期中。

## 本章目标

读完本章，你将能够：
1.  理解 Agentic Coding 背后的原理（如 Stop Sequences 和 Context Window）。
2.  掌握主流工具（Cursor, Claude Code）的高级用法。
3.  构建自己的 AI 编码工作流，从"写代码"转型为"编排 AI"。
